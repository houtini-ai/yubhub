feat: Add comprehensive all-feeds dashboard with modern React UI

Implemented new `show_all_feeds_dashboard` MCP tool that displays all feeds in a sortable, modern interface:

**New Files:**
- src/dashboard-generator.ts: Self-contained React dashboard generator using CDN imports (React, Tailwind, Babel)

**Features:**
- Modern dark theme UI with Tailwind CSS
- Sortable table columns (name, total, enriched, failed, success rate, last run)
- Summary statistics cards (total feeds, jobs, enriched, failed)
- Success rate indicators with color coding
- External links to careers pages
- XML feed links (disabled for empty feeds)
- Responsive grid layout
- Inline SVG icons (lucide-react style)
- Real-time relative timestamps

**Technical Approach:**
- Uses React 18 via CDN (unpkg) for artifact compatibility
- Babel standalone for JSX transformation
- Tailwind CSS via CDN for styling
- Self-contained HTML - no build step required
- Parallel API requests to fetch all feed stats
- Graceful error handling for failed feed fetches

**Why Not Template System:**
- Modern UI kits (shadcn/ui) require npm install, not suitable for artifacts
- Self-contained React with CDN imports provides better visuals
- No template maintenance needed
- Works directly in Claude artifacts and MCP UI resources

**Usage:**
Call `show_all_feeds_dashboard` with no parameters to see comprehensive dashboard
